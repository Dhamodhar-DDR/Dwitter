# Dwitter
* Full stack web application using MERN. 
* Deployed on : [Dwitter](https://dwitter0.herokuapp.com)

# Setup and Run Locally
```shell
npm install
cd client/ && npm install && npm run build
node server.js
```
* The app runs locally on http://localhost:8080
